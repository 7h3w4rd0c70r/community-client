Welcome to LAMPA!

LAMPA is a powerful, yet very user-friendly, set of tools for teachers, students and language schools. Once you are all set up, it will save you time, it will facilitate lesson continuity, and it can help you supplement you income.

Step 1. To use LAMPA to its full potential, you must set up six basic elements in the SET UP menu:

  1. Set up your school by checking or editing the information LAMPA has already filled in using your registration information and add the following: School year starts on, School year ends on, Lesson duration

  2. Name and invite teachers

  3. Define the various locations where you expect lessons to take place, e.g. room numbers, firms,workstations for online lessons, etc.

  4. Define the courses that you intend to teach

  5. Designate your groups, i.e. individuals or classes taking a course at a given time and place

  6. Name and invite your students by entering their names and emails

N.B. When setting up all other users, i.e. teachers, students or clients, it is important that you enter their email. LAMPA uses email addresses as unique user I.D.s, therefore this is essential in creating the user account and no two users may share the same email address.

Step 2. After you have set up your basic elements, you can start adding lessons using the Add feature in the MY LESSONS menu. Click on Add and do the following:

  1. Select the Group you would like to add to your timetable. If you have forgotten to add the group, no worries, you can add a new group using the Add Group button on the right.

  2. Use the calendar icon to select the date on which you would like to begin the course or add a single lesson.

  3. Choose an End Date for the course or last lesson, or select By Lesson Count if you would prefer to designate a specific number of lessons to be taught without a concrete end date.

  4. Click Add Lessons and designate the day of the week, the start time, the duration, Teacher and Location.

  5. Repeat if the group is to have more than one lesson per week.

  6. After all lessons for the group have been listed, click on Generate Lessons

  7. Check the list LAMPA has generated.If there are mistakes or lessons you wish to remove, you can un-check the lessons using the check boxes at the right

  8. Click on Save Lessons

  9. Click on the MY LESSONS menu and open a timetable to check them

  10. Repeat to add lessons for other groups.

The more teachers and students who use the system based on your initial invitation, the more you can supplement your regular teaching income. To invite teachers so that they can start using LAMPA’s tools for free, do the following:

  1. Click on INVITE PEERS

  2. Enter the teacher’s e-mail address

  3. Personalize the invitation message

  4. Click on Send

How can LAMPA supplement a partner’s income?

LAMPA provides mobile learning games, personal dictionaries and course tracking tools for students. When students download the app and buy a 1-year license, a portion of the purchase price goes directly back to the teacher or school that has invited the student; another portion goes back to the teacher or the school that invited that teacher, and so on. This is repeated on five levels. Simply, the more new users you invite, the more you can earn.
